Welcome to the Minty CSS framework

It's a framework for people who dont want one

The goal of Minty is to jumpstart your static site with this CSS framework / HTML Template, while keeping small but very organized code. Downloading the Minty framework will give you this website, which is made of:

HTML Boilerplate

Bare minimum html to get you started, meta tags, body, IE conditional, and this page


12 Column Grid

Mobile friendly simple grid that works on every browser (even evil IE).


Pre-Organized Code

The CSS files have a table of contents so you can code in organized sections.


Normalized Styles

Body and font formatting to keep things predictable.

12 Column grid example, class naming organization is ▸ row ▸ column ▸ column-size

The media query is bare bones, desktop and mobile. Specify the column width by size:

Desktop: c12, c6, c4, etc. 12 is full width, 6 is 50%, and so on.

Mobile: s12, s6, s4, etc. 12 is full width, 6 is 50%, and so on.

The last row is c3 for desktop and s6 for mobile, resize to see.
