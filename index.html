Welcome to the Minty CSS framework

It's a framework for people who dont want one

The goal of Minty is to jumpstart your HTML and CSS project with a super light, simple, and tested solution. Starting with the Minty framework will provide a few things, literally.


HTML Boilerplate

Bare minimum html to get you started, meta tags, body, IE conditional, and this page


12 Column Grid

Bootstrap style, simple grid that works on every browser (even evil IE).


Pre-Organized Code

The CSS files have a table of contents so you can code in organized sections.


12 Column grid example, class naming organization is container ▸ row ▸ col ▸ col-size

There are 2 media queries, med and sml. To specify mobile size, use sml class following the grid.css sizing.
